Задание «RestFULL API»

клонирование репозитория:

1. git clone git@github.com:decaxp/RESTFULL-API.git

Полный путь: https://github.com/decaxp/RESTFULL-API.git


2. Импортировать базу данных restfull.sql

4. Profit!
